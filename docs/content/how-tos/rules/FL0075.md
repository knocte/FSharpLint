# FavourNonMutablePropertyInitialization (FL0074)

*Introduced in `0.21.1`*

## Cause

Rule to detect mutable property initialization.

## Rationale

It helps make code readible.

## How To Fix

Replace all occurrences of mutable property initialization with non mutable property initialization

## Rule Settings

    {
        "FavourNonMutablePropertyInitialization": { "enabled": false }
    }
